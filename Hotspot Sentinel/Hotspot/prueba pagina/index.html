Proyecto SAR — Análisis Multibanda y Multipolarización

Un informe profesional (plantilla) en español para analizar datos SAR de distintas frecuencias y polarizaciones.

Resumen

Como Alicia atravesando el espejo, este proyecto utiliza datos de radar de apertura sintética (SAR) en distintas frecuencias y polarizaciones para revelar procesos físicos en la superficie terrestre. La plantilla guía al investigador desde la descarga de datos hasta métodos básicos de procesamiento y visualización, e incluye scripts de ejemplo reproducibles.

Área de estudio (ejemplo)

Ejemplo propuesto: Delta del Ebro (Cataluña, España). Se trata de una zona de transición terrestre-marina con humedales, canales, vegetación y actividad humana. Es ideal para comparar respuestas en C-band vs L-band y polarizaciones (VV/VH o HH/HV).

Datasets recomendados

  • Sentinel-1 (ESA) — Banda C, polarizaciones VV y VH (descarga gratuita vía Copernicus/SciHub o AWS).
  • ALOS PALSAR (JAXA) — Banda L, útil para penetración de dosel y detección de suelo.
  • UAVSAR / NISAR (NASA/ISRO) — opciones L-band/multifrecuencia cuando estén disponibles.

Busque imágenes con múltiples polarizaciones (por ejemplo VV/VH o HH/HV) y varias fechas para análisis temporal.

Metodología (resumen)

  1. Descargar productos SAR calibrados (sigma0 o gamma0) en GeoTIFF.
  2. Convertir a dB: 10 * log10(sigma0) si es necesario.
  3. Aplicar filtros espaciales/uplift (opcional) y alinear bandas (reprojectar/resample).
  4. Calcular índices comparativos (p. ej. ratio HV/HH, diferencia L/C en dB).
  5. Clasificación simple: detección de agua por umbral, detección de cambios entre fechas.
  6. Visualizar: composiciones RGB, mapas web (folium) e incluir capas auxiliares (vectores, ortofotos).

Hipótesis (ejemplos)

  • Las zonas inundadas aparecerán con valores de retrodispersión muy bajos en todas las polarizaciones de C-band; L-band podría mostrar mayor señal en vegetación sumergida.
  • En bosques densos, L-band (ALOS) mostrará mayor retrodispersión debida a penetración en el dosel, mientras que C-band refleja principalmente la copa.
  • Las diferencias VV/VH (o HH/HV) resaltarán estructuras verticales (árboles, infraestructura) frente a superficies lisas (agua, suelo desnudo).

Visualizaciones sugeridas

  • Composición RGB: asignar bandas/fuentes (por ejemplo L-band -> R, C-band -> G, ratio -> B).
  • Mapa de detección de agua por umbral en dB y comparación temporal.
  • Mapa web interactivo (folium) con controles para alternar capas (C-band, L-band, óptico).

Los scripts de ejemplo incluyen un flujo de trabajo mínimo para crear estos productos.

Mapa interactivo (ejemplo)

El mapa a continuación se activa si dispone de una conexión a internet y si se sirven capas raster (tiles) o WMS. Puede usar gdal2tiles o rio-tiler para crear tiles desde GeoTIFFs SAR.

Controles: cambie la capa base o agregue una URL XYZ/WMS en el README para ver su ráster.
